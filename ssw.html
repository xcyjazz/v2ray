在 Windows设备上使用 V2rayN

Windows

系统要求： Windows 7 X64 及以上
软件版本：V2rayN
设备要求： Windows PC / Tablet

1. 使用浏览器登录到 网站 管理门户

建议使用 Chrome 访问网站客户中心，然后找到我的产品，点击您的套餐进入配置详情页面


2. 获取订阅链接

点击页面中的「订阅地址」功能区下方按钮，然后在弹出的窗口中复制URL地址。


3. 连接使用


打开 v2网络加速器 程序，然后登陆网站账户,选择对应V2套餐后，选择您需要的节点，点击下方「连接」按钮即可。
